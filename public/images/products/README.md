# Product & hero images

Drop real photos here, then reference them from `lib/products.ts` via the
`image` field, e.g.:

```ts
image: "/images/products/royal-taj-masterpiece.jpg",
```

Recommended: square or 4:3 photos, ~1000px wide, well-lit on a clean
background. Until an image is added, a styled claymorphic placeholder is shown
automatically.
